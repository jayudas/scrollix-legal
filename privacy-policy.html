Last Updated: February 7, 2026

1. Introduction

Scrollix LLC ("we," "our," or "us") operates the Scrollix desktop application (the "App"). This Privacy Policy explains how we collect, use, disclose, and safeguard your information when you use our App.

By using Scrollix, you agree to the collection and use of information in accordance with this policy.

2. Information We Collect

We collect the following types of information:

a) User Account Information

  • Email address (used for login)
  • Password (hashed using PBKDF2-SHA256 - never stored in plain text)
  • First name and last name (optional)
  • Onboarding completion status
  • Active social account selection

b) Instagram Account Information

  • Instagram username
  • Instagram user ID
  • Instagram profile information (name, profile picture URL, biography)
  • Instagram Business account status
  • Last token refresh timestamp

c) Instagram Content

  • Posts you create in Scrollix (captions and hashtags)
  • Images and videos you upload
  • Scheduled post dates and times
  • Post publishing status and timestamps
  • Video duration and thumbnail images

d) Instagram API Access Tokens

  • OAuth access tokens (encrypted using Fernet encryption)
  • Token expiration dates
  • OAuth refresh tokens (encrypted)
  • Token scopes granted

Instagram permissions we request:

  • instagram_business_basic - Access to your Instagram Business account profile
  • instagram_business_content_publish - Ability to publish posts to your Instagram account

e) Password Reset Information

  • Password reset request timestamps
  • Hashed reset tokens (SHA-256)
  • Token expiration times (1 hour from creation)
  • Token usage status (whether token has been used)

f) Usage Data

  • App usage statistics
  • Error logs and diagnostic information
  • Features you use

g) Device Information

  • Operating system and version
  • App version
  • Device identifiers (for error tracking only)

5. Data Storage and Security

a) Storage Locations

  • Local Storage: Your data is stored in a secure local SQLite database on your device (~/Library/Application Support/SMS/sms.db on macOS)
  • Temporary Files: Uploaded media files are temporarily stored locally before publishing, then automatically deleted after successful publish
  • Cloud Storage: Images and videos are stored in YOUR Google Cloud Storage bucket (you provide your own bucket - we do NOT have access to it)

b) Security Measures

  • Password Protection: Passwords are hashed using PBKDF2-SHA256 algorithm (never stored in plain text)
  • Token Encryption: OAuth tokens are encrypted using Fernet encryption (AES-128-CBC with HMAC-SHA256 authentication)
  • HTTPS Encryption: All external API communications use HTTPS encryption
  • Secure File Permissions: Local database and configuration files have restricted permissions (owner read/write only)
  • Regular Security Updates: We provide app updates with security patches

c) Data Retention

  • We retain your data as long as you use the App
  • You can delete your account and all associated data at any time
  • Temporary uploaded files are automatically deleted after successful publish
  • Password reset tokens expire and are deleted after 1 hour
